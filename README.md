# Casino Niagara Poker Website

Static site output for https://casinoniagarapoker.com.

## Structure

- `index.html` — poker room + live status + bad beat summary
- `calendar.html` — tournament calendar + recent results
- `badbeat.html` — bad beat tracker
- `stats/*/index.html` — yearly and main-event stats pages
- `data/*.json` — published data consumed by frontend JS
- `js/` — client scripts
- `css/` — shared + page-specific styles

## Data pipeline

Generated by scripts in `../code/` and published via cron jobs.
Frontend reads JSON from `website/data/` only.

## Notes

This directory is its own git repo and should be committed/pushed independently.
